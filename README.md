# UDAYRA

Udayra is a GST preparation and compliance-readiness platform for CA firms.

## What UDAYRA does
- Organization invoices upload from any software (tally, vyapar, local apps)
- Prepares GST-ready summaries (read-only)
- Reduces CA follow-ups  and last-minute stress

## What Udayra does NOT do
- Does not file GST returns
- Does not login to GST portal
- Does not replace CA judgment 

> UDAYRA PREPARES.CAs review and file.
